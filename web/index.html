    Malayalam Sign Language Recognition System

    The main goal of this project is to help the deaf-mute community in Kerala by using the newly developed Malayalam sign language.

    MSLRS is a real-time platform-independent software that is able to recognize the newly published Malayalam Sign language hand gestures.
